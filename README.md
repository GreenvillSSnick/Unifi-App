# Unofficial UniFi Desktop App

Unofficial UniFi Desktop App for Windows, macOS, and Linux.

## Features

- Custom Top Bar
- Theme Support
- Persistent Storage

## Installation

```bash
npm install
npm run build
```

This will create a `release` folder with the executable file for setup.

## Usage

```bash
npm start
```

## Why did I make this?

I was using the UniFi website, and I just thought, why isn't this a desktop app? So I decided to make one. I do access the UniFi website very often, and I just thought it would be nice to have a desktop app for it.
